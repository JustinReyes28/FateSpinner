# FateSpinner

A simple "Spin the Wheel" game where you can add your own words to the wheel and spin it to get a random result.

## How to Use

1.  Open the `index.html` file in a web browser.
2.  Enter a word in the input field and click the "Add Word" button.
3.  Add at least two words to the wheel.
4.  Click the "Spin" button to spin the wheel.
5.  The winner will be announced in an alert box.

## Files

*   `index.html`: The main HTML file.
*   `style.css`: The CSS file for styling the application.
*   `script.js`: The JavaScript file containing the game logic.
